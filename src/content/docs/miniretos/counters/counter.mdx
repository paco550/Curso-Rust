### Contador (JS) 

## React: Creando un Contador con useState

#### vamos a crear un componente de React que actúa como un contador. Utilizaremos el hook `useState` para gestionar el estado del contador.

## Paso 1: Importar useState

#### Primero, necesitamos importar el hook `useState` desde React:

```jsx
import { useState } from 'react';
```
#### El hook useState nos permite agregar estado a nuestros componentes funcionales.
#### Paso 2: Definir el Componente Counter 
#### A continuación, definimos un componente funcional llamado Counter:

```jsx
function Counter() {
  // Declara una variable de estado llamada "count" y una función para actualizarla llamada "setCount"
  const [count, setCount] = useState(0);
```
```
Aquí, estamos utilizando useState para declarar una variable de estado llamada count y una función para actualizarla llamada setCount. useState toma un valor inicial, en este caso 0.
```
```
Paso 3: Renderizar el Contenido del Componente
Dentro del componente, vamos a devolver un div que contiene el valor actual del contador y dos botones para incrementar y decrementar el contador:
```
```jsx
Copiar código
  return (

      {/* Muestra el valor actual del contador */}

      {/* Botón para decrementar el contador */}

```
{count} dentro de <h2> muestra el valor actual del contador.

Cada botón tiene un evento onClick que llama a setCount para actualizar el valor de count.
